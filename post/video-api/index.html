各大短长视频网站解析接口

2022-03-22 Views API | 接口 | 视频 | 解析296字2 min read繁

接口调用说明

本播放器接口有两种调用方式，一种是有P2P加速（使用CDNBye跨平台P2P流媒体加速引擎）反之另一种没有，P2P加速是一种点对点的传输技术，数据是从不同的用户之间共享传输的，在线人数越多，速度越快。但是如果对于解析其它平台视频播放可能出现不稳定，所以本人推荐在解析其它平台视频时不要用P2P加速模式就用普通解析接口就行了，而在播放“.m3u8”等视频时可以使用P2P解析接口

普通解析接口

https://player.iuk.ink/m3u8.php?url=

P2P加速接口

https://player.iuk.ink/m3u8.php?p2p=true&url=

苹果CMS播放器代码

MacPlayer.Html = '<iframe src="https://player.iuk.ink/m3u8.php?p2p=true&url='+MacPlayer.PlayUrl+'" height="100%" width="100%" frameborder="0" scrolling="no" allowfullscreen="true" seamless></iframe>';
MacPlayer.Show();

进入苹果CMS后台，操作流程：视频→播放器→编辑播放器→播放器代码→填入 编码 里保存即可
注意，播放器需要禁用解析状态，才能生效。
