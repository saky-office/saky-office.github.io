使用教程

CID：bafybeialywlutdkpfshhih5qvatoxav75apbt4idlmqejfgnhq47gpo4nq

  1. 运行服务端程序
    运行run.bat,出现3个不同颜色的终端（请不要关闭此窗口，后台运行）
  2. 全局命令（请在米色终端内输入命令）
# 创建游戏账号
account areate xxx[用户名] 10000 [自定义UID 可省略]   # 例：account areate moe 100001
permission add 用户名 *  # 例：permission add moe *
  1. 管理员运行私服连接器
  2. 进入原神客户端，变成国际版登录页面即为链接成功，输入你刚刚创建的账号，首次输入的密码即为账户密码

游戏内命令

找到好友service对话框里面输命令,常用命令如下：

添加怪物 /spawn [怪物ID] [等级] [数量]

获取道具 /give [道具ID] [数量] [等级]

获取角色 /givechar [角色ID] [等级]

掉落道具 /drop [道具ID] [数量]

删除道具 /clearartifacts 删除所有未装备及未解锁的圣遗物 
删除道具 /clearweapons 删除所有未装备及未解锁的武器

世界等级 /setworldlevel [等级] 重新登录查看效果

好感等级 /setfetterlevel 当前角色  

重置命座 /resetconst 重置当前角色的命座，重新登录查看效果
全体回血 /heal

移动端🔗教程

  1. 准备原神国际版客户端
  2. 安装两个证书一个CA证书（手机设置，大概率在安全里），一个wifi证书（wifi设置页）
  3. 修改代理为手动，用流量的在设置数据页，用wifi的在连接的wifi页
  4. 主机 , 端口填写自己搭建的服务端IP和端口
  5. 数据异常和网络错误的自己查上边2条
  6. 账号验证失败的大退重进
