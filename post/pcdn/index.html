家庭 PCDN 选择（京东云、甜糖、网心云、超点云、猕猴桃）

2022-04-24 Views PCDN | 京东云路由 | 网心云780字3 min read繁

京东云

推荐程度：⭐️⭐️⭐️⭐️⭐️

  • AX1800【适合200m以上宽带】：https://u.jd.com/ZwfmMIp
  • 鲁班【1年包回本，适合200m以下宽带】：https://u.jd.com/ZtfhP2q

优点：无需公网IP，大厂靠谱，可以当路由器用？
缺点：需要购买专用设备，部署周期长达7天

猕猴桃

推荐程度：⭐️⭐️⭐️⭐️⭐️

账号注册

优点：

  1. 支持多线汇聚，网络只需要设置好upnp即可（多线用户还需要设置端口分流）。
  2. 一台设备只需要开一个猕猴桃，会自动根据磁盘容量多个进程（350g/300g一个进程），榨干你的硬盘和带宽。
  3. 上游调度不错，带宽利用率高，部署快（尤其是多进程），一天就能跑量。
  4. 一个进程（350g/300g一个进程）就能跑满30m上行，适合有闲置机械硬盘的用户 。
  5. 安装方便，全自动设置，官方设置的很好，部署基本不会遇到疑难杂症。

缺点：

  1. 没有app，只有微信小程序，功能比较简陋
  2. 知名度较低
  3. 相关教程较少

网心云

推荐程度：⭐️⭐️⭐️⭐️

账号注册

优点：

  1. 拥有功能丰富的app，详尽的视频、图文教程，迅雷旗下产品
  2. 上游调度不错，带宽利用率高。
  3. 支持pppoe拨号，支持多线汇聚，省去路由的成本

缺点：

  1. 部署周期长达7天，这七天收益非常低。
  2. 部署需要购买激活码 (有些不需要)
  3. 相对别家cdn，带宽单价较低。

甜糖

推荐程度：⭐️⭐️⭐️

  • https://tiptime.cn/ 邀请码: 488153

优点：

  1. CPU占用极低
  2. docker快速部署，只要能运行docker就可以部署。
  3. 内存仅需1g，部署多个只需要无脑加内存就可以了。
  4. 无需考虑磁盘性能，一个16g优盘便可轻松跑满30m上行，家里有闲置优盘，又不想购买ssd等跑量的渣友比较好的选择。
  5. 网络连接数非常小（只有两位数，其他cdn都是2000起步），对路由器负担很小。
  6. 部署一天就能跑量。
  7. 背靠大厂：360

缺点：

  1. 由于缓存过小，上游业务不足，实际调度起伏较大，不容易带宽跑满，有时甚至一天都没有收益。
  2. 就算使用大容量ssd，也不会缓存大量数据装满ssd，对跑量帮助性价比较低，不如优盘划算。对于有闲置固态，机械硬盘的渣友不如别家cdn划算。
  3. 网络部署不支持upnp，需要自行设置端口转发或DMZ。
  4. 不支持多线汇聚，需要为甜糖设置专用线路。
  5. 经常检查网络类型失败

超点云

推荐程度：⭐️⭐️
优点：

  1. 上游调度不错，带宽利用率高，单价不错。
  2. 部署简单，支持pppoe拨号，只需要开启upnp/dmz就能跑。
  3. 一天就能跑量。
  4. 官方推荐500g以上硬盘跑，小容量硬盘缓存满了收益会降低。

缺点：

  1. 50元才能开始提现
  2. 一个超点云只能使用一条线路，多线渣友需要部署多台超点云

    • 猕猴桃
